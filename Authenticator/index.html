authenticate(_:​) 

func authenticate(_ onChange: @escaping (Result<Secret, Error>) -> Void)

Return a Secret and store it in storage.

Always call Secret.store with storage when receiving the Secret .

Using TransientStorage as Storage allows to disregard any storing mechanism.
